### Setup

- Step 1: `pod install`
- Step 2: Setup Mapbox token
  - Option 1: https://docs.mapbox.com/ios/api/maps/4.4.1/infoplist-keys.html
  - Option 2: https://docs.mapbox.com/help/troubleshooting/private-access-token-android-and-ios/
- Step 3: Enjoy!
